## 🎯 Obiettivo

Entrare nel Raspberry tramite SSH 🖥️ ed usare SOLO i comandi base della shell Linux per:

⚔️ attaccare gli altri studenti
🛡️ difendere la propria cartella
👻 nascondere le proprie tracce
🏆 ottenere il punteggio più alto possibile



## 🚪 Accesso all’Arena

Dopo l’accesso **SSH** al **Raspberry Pi** 🥧 dovrete entrare nella vostra directory personale usando:

```bash
cd ./<classe-data>/<classe-cognome-nome>
```

Da quel momento…
🔥 **La Battaglia avrà Inizio** 🔥

## 🧰 Comandi Consentiti

* `cd`
* `ls`
* `mkdir`
* `rm`
* `echo`
* `>`
* `>>`

## ⚔️ Come Attaccare

Un attacco consiste nello scrivere il proprio:

```text
cognome-nome
```

all’interno dei file presenti nelle cartelle degli altri studenti 📂

💡 Più volte compare il vostro nome nei file avversari, più punti otterrete!

**Esempio**:

```bash
echo "rossi-mario" >> file.txt
```

## 🧠 Stealth Mode 👻

### ⚠️ Gli attacchi devono essere Nascosti!

Se il nome del file è troppo riconducibile all’attaccante 😬 (esempio: `.rossi`, `mario.txt`, `rossi_attack` ecc.)

➡️ verrà applicata una penalità.


## 🏆 Griglia Punteggi

### ⚔️ Attacco

file

* Nome file NON appropriato ❌ >> 0 pt
* File appropriato 📄 >> +1 pt
* Dotfile appropriato 👻 (`.nomefile`) >> +2 pt

📌 Ogni comparsa del vostro nome conta come un attacco.


## 🛡️ Difesa

Ogni file presente nella vostra cartella personale abbasserà il vostro punteggio finale 😵

* File normale 📄 >> -1 pt
* Dotfile 👻 >> -2 pt

### ⚠️ Anche i file nascosti contano!

## 🚫 Attenzione

* ❌ Non cancellare cartelle
* ❌ Non usare nomi volgari o non appropriati
* ❌ Non uscire dall’area di gioco
* ❌ Non sabotare il sistema

📌 I nomi dei file saranno valutati dal docente.

Se considerati non appropriati o non validi, gli attacchi NON verranno conteggiati.

## 🏅 Classifica Finale

Il punteggio totale sarà dato da:

```text
Attacco - Difesa
```

🔥 I giocatori con il punteggio più alto riceveranno anche un voto positivo per impegno, rapidità e strategia!
